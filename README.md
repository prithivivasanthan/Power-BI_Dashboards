
# Data Job Dashboard w/ Power BI

## Overview
This project is an interactive Power BI dashboard built to explore trends in the data job market.  
It uses a dataset of around **479K job postings** to analyze demand, salaries, and hiring patterns across different roles.

The goal was to turn raw job data into something more clear and useful through visual storytelling.

## What the dashboard covers

- Overall job count and salary benchmarks  
- Job trends across 2024  
- Salary comparison (yearly vs hourly)  
- Demand for different roles  
- Global job distribution  
- Hiring platforms used by companies  

---

## Key insights

- Data Engineer roles show the highest demand  
- Median salary is around **$113K per year**  
- Most jobs are full-time  
- LinkedIn is the most commonly used hiring platform  
- Job postings drop toward the end of the year and slightly recover  

---

## Drill-through analysis

A drill-through page is included to explore role-specific insights.

For example (Business Analyst):
- Salary breakdown (yearly and hourly)  
- Work-from-home percentage  
- Degree requirement trends  
- Health insurance availability  
- Job locations across regions  
- Platform-wise job distribution  

---

## Tools and techniques used

- Power BI  
- Power Query (data cleaning and transformation)  
- DAX (calculated measures and metrics)  

---

## Limitations

- Data is not real-time  
- Some roles or regions may have limited data  
- Salary values are aggregated estimates  

---

## Possible improvements
